# 💡 IdeaFlow — Collaborative Group Idea Board

## 🚀 Live Features

- **Modern Dashboard UI:** Built with a "glassmorphism" approach, featuring soft shadows and high-end card elevation.
- **Dynamic Theme Engine:** Integrated Dark and Light modes with a smooth transition and eye-comfortable mesh gradients.
- **Team Onboarding:** Dedicated workflow to register members before assigning ideas.
- **Advanced Typography:** - **Headings:** Bold Inter typeface for structural clarity.
  - **Metadata:** Author names are styled with **italicization and 0.7 opacity** to create a refined visual hierarchy.
- **Persistence:** Full `localStorage` integration ensures all members and ideas remain available after a page refresh.
- **Real-time Analytics:** Automated counter tracks the total number of ideas in the current session.
- **Interactive Feedback:** Scale-down effects (`scale-95`) on click and hover transitions on all interactive elements.

## 🛠️ Tech Stack

* **HTML5:** Semantic markup for accessibility and SEO.
* **Tailwind CSS:** Utility-first styling for a responsive, modern interface.
* **Vanilla JavaScript:** Lightweight ES6+ logic for state management and DOM manipulation.
* **Google Fonts:** Utilizing the 'Inter' font family for a professional, clean look.
